## Installation

```bash
git clone https://github.com/AleksandraWeber/todo-app.git
or
git clone git@github.com:AleksandraWeber/todo-app.git
cd todo-app
npm install
```

## Usage

To run app in **dev** mode

```bash
npm start
```

The app should be running on `http://localhost:3000/`


<hr/>

Prepare build version

```bash
npm run build
```
